gmail.py : the main script were all the other functions are called

read.py : Reads as many emails as the user passes as argument(currently the max is set to 20 in order not to overflowd the system).

send.py : Creates and email through converting string text to binary and sends it .Arguments:
		-Message in string format.
		-Email destination in string format.
		-The subject of the email.

Google.py : Creates a token in order to connect to Google's api services.

credentials.json : The credentials of my private key.
