__init__.py : exists so the interpreter knows that the content of this folder is importable

googleMapsDirections.py : contains a function that opens a Google Maps Directions Page, having already filled some elements, depending on the parameters passed.

googleMapsSearch.py : contains a function that opens a Google Maps Search Page. The parameter passed is used as the search content. (if no parameter is passed, then the page opens with an empty search bar)

openUrls.py : contains a function that is used to open URLs, using the system's default browser. (is exists, for the purpose of avoiding to repeat the same code multiple times)
